# Run Peacth-AC API

Instalar requerimientos
```
pip install -r requeriments.txt
```

Se requiere tener una base de datos en mongoDB
```
use peacth-ac
```

Borrar "0001_initial.py"

```
python3 manage.py makemigrations api
python3 manage.py migrate

```
